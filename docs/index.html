Automatize o Seu Fluxo de Trabalho

Integre com o seu pipeline de CI/CD. Construa e implemente automaticamente a sua documentação a cada commit, garantindo que está sempre atualizada.

Configurar CI/CD → 
name: Deploy Docs
on:
  push:
    branches:
      - main
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Deploy to Pitchutcha
        uses: pitchutcha/deploy-action@v1
        with:
          token: ${{ secrets.PITCHUTCHA_TOKEN }}
