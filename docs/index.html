Query Logs

Get deeper insights by writing SQL queries on function logs. 

SELECT 
  status_code,
  COUNT(*) AS count,
  AVG(execution_time) AS avg_time
FROM function_logs
WHERE created_at > NOW() - INTERVAL '1 hour'
GROUP BY status_code
ORDER BY count DESC;

Results:

status_code count avg_time
200 847 142ms
201 231 284ms
500 12 892ms
